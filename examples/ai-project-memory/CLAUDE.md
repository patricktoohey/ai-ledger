# Project Instructions

## At the Start of Every Session
- Read plan.md and status_update.md before doing anything
- Summarize current state and confirm the next step with the user

## During Work
- Follow the steps defined in plan.md
- After completing each major step, update status_update.md
- Save all outputs to the /outputs folder with descriptive filenames

## Rules
- Never modify files in data/raw/
- Do not send sensitive data to external services
- Document any issues or edge cases in status_update.md
- If unsure about a decision, ask before proceeding
